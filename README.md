**Linux Security & DevSecOps Lab**

**Overview**

This repository documents a hands-on Linux security and DevSecOps lab where I reproduced and mitigated ten real-world operational scenarios affecting filesystem integrity, access control, logging, deployment, and process management.
All commands, observations, explanations, and screenshots are consolidated in documentation.md.

**Scenarios Covered**

1.	Shared Document Deletion
2.	Log Overwrite & Forensic Loss
3.	Permission & Ownership Drift
4.	Relative Path Deployment Failure
5.	Monitoring Failure After Log Cleanup
6.	Sensitive Data Exposure Hunt
7.	Shared Directory Stability Controls
8.	Special Permission Risk Review
9.	Process Anomaly Investigation
10.	Incident Documentation via Heredoc

Lab Environment: OS: Kali Linux VM
